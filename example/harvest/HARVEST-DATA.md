# HARVEST-DATA — 内容数据文档

> HARVEST 的文案与内容资产。这些数据应与实现代码分离，便于非技术人员编辑。

---

## 1. 空状态引导文案

当用户还没有创建任何习惯时，页面显示以下内容：

### 1.1 主标题

```
还没有习惯
```

### 1.2 说明文字

```
点击下方的「管理习惯」，添加你本周想要追踪的日常。
最多可以设置 7 个。
```

---

## 2. 界面固定文案

### 2.1 导航与操作

| 位置 | 文案 |
|------|------|
| 管理习惯入口 | 管理习惯 |
| 回到本周按钮 | 回到本周 |
| 周导航左箭头 | ‹ |
| 周导航右箭头 | › |

### 2.2 管理习惯面板

| 位置 | 文案 |
|------|------|
| 面板标题 | 管理习惯 |
| 新增输入占位符 | 例如：早起、阅读、运动 |
| 添加按钮 | 添加 |
| 删除按钮提示 | 删除 |
| 关闭按钮 | 完成 |
| 习惯数量提示 | {current} / 7 |

### 2.3 日期表头

一周的日期缩写（表头）：

```
周一  周二  周三  周四  周五  周六  周日
```
